# Demo Block Analysis

## Task Description
Create a new "demo" block containing three author inputs: image, title, and description.

## Requirements
- **Image**: An author-provided image (required)
- **Title**: A heading for the demo item (required)
- **Description**: A paragraph of descriptive text (required)
- Simple, single-item layout
- Responsive across all viewports

## Acceptance Criteria
1. Block renders an image, title (heading), and description (paragraph)
2. Layout is responsive: stacks vertically on mobile, horizontal on tablet/desktop
3. Image displays properly at all viewports
4. No console errors
5. Linting passes
6. Accessible (proper heading hierarchy, alt text on image)
